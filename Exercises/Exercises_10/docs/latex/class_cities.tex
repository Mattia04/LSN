\doxysubsection{Detailed Description}
represents a collection of cities. 

The \doxylink{class_cities}{Cities} class contains a vector of \doxylink{class_city}{City} objects, and a vector of possible UIDs (the UIDs of the cities in the collection).

The \doxylink{class_cities}{Cities} class contains a map that associates each possible UID to the index of the city in the vector, to optimize data retrieval.

The \doxylink{class_cities}{Cities} class also contains a distance matrix, which contains the precomputed distances between all cities for optimization purposes.

The \doxylink{class_cities}{Cities} class is designed to be immutable.
