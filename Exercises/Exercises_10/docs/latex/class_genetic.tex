\doxysubsection{Detailed Description}
represents a genetic algorithm for solving the traveling salesman problem. 

The class manages a population of chromosomes, where each chromosome represents a possible solution to the TSP. The class provides methods for evolving the population through selection, crossover, and mutation. The parameters for mutation and crossover probabilities can be set in the constructor, and are stored as member variables.

The fitness of each chromosome is calculated based on the total length of the route it represents, using the L1 distance. The class also provides methods for retrieving the best solution and the average fitness of the best half of the population.

The class is designed to work alongside the \doxylink{class_cities}{Cities} class, which provides the necessary information about the cities and their distances.

void Genetic::\+new\+\_\+generation (\begin{DoxyParamCaption}\item[{const \doxymbox{\hyperlink{class_cities}{Cities}} \&}]{map}{, }\item[{const double \&}]{exponent}{, }\item[{\doxymbox{\hyperlink{class_random}{Random}} \&}]{rnd}{}\end{DoxyParamCaption})}



creates a new generation of chromosomes. The new generation is found by:\+ 


\begin{DoxyEnumerate}
\item Sorting the chromosomes based on their fitness (best fitness first).
\item Selecting two random parents from the current population using the selection method discussed below.
\item Recombining the selected parents to create two offspring using the crossover method.
\item Mutating the offspring.
\item Repeating from step 2 until the new population is the same size as the original.
\item Replacing the old population with the new one.
\item Recomputing the fitness of the new population.
\end{DoxyEnumerate}

The selection method is based on the fitness of the chromosomes. The individuals are selected using the formula:\+ \[j = \mathrm{population size} \cdot \mathrm{floor}(r ^ p)\] where $r$ is a random number in the range \+[0,1) and $p$ is an exponent that can be tuned to increase or decrease the selection pressure. Since the fitness is sorted from best to worst, a higher value of $p$ will increase the selection pressure.
