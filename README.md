# Laboratorio di Simulazione Numerica (LSN)

This repository contains my solutions to the assignments for the course *"Laboratorio di Simulazione Numerica"* ("Numerical Simulation Laboratory") at the University of Milan (UNIMI), academic year 2025-2026.

## Repository Structure

The full text of the exercises can be found in the `Numerical Exercises` directory.

The simulation code is located in `Exercises`. Each assignment is organized in a folder named `Exercises_XX`, and each individual exercise could be contained in its own subfolder `Exer_XX_YY`.

Data analysis was performed using Jupyter Notebooks, which are stored in the related `Exercises_XX` directory.

### Arguments of the exercises

Below is a brief overview of the topics addressed in each assignment:

- Exercises 1: Monte Carlo methods and data blocking.
- Exercises 2: Importance sampling and random walk.
- Exercises 3: Stochastic integration and vanilla option pricing.
- Exercises 4: Energy-conserving ODE solvers and molecular dynamics.
- Exercises 5 (optional): Hydrogen wavefunction simulation.
- Exercises 6: Ising Model and Markov Chain.
- Exercises 7: Molecular Dynamics and tail corrections.
- Exercises 8: Variational Principle and Simulated Annealing.
- Exercises 9: Travelling Salesman and Genetic Algorithm.
- Exercises 10: Code Parallelization.
- Exercises 11: Neural Networks.
- Exercises 12: Convolutional Neural Networks.
