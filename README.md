# Laboratorio di Simulazione Numerica (LSN)

This repository contains my solutions to the assignments for the course *"Laboratorio di Simulazione Numerica"* ("Numerical Simulation Laboratory") at the University of Milan (UNIMI), academic year 2025-2026.

## Academic Integrity Notice

This repository is shared for learning purposes only.

If you are enrolled in the "Laboratorio di Simulazione Numerica" course (or similar),
you are expected to write your own solutions. Submitting this code, in whole or in part,
as your own work is likely to violate your institution’s academic integrity policies.

Use this material responsibly.

## Repository Structure

The full descriptions of the exercises can be found in the `Numerical Exercises` directory.

The simulation code is located in `Exercises`. Each assignment is organized in a folder named `Exercises_XX`, and each individual exercise is contained in its own subfolder `Exer_XX_YY`.

Data analysis was performed using Jupyter Notebooks, which are stored in the `Notebooks` directory.

The `Media` folder contains images and videos generated from the simulations.

### Arguments of the exercises

Below is a brief overview of the topics addressed in each assignment:

- Exercises 1: Monte Carlo methods and data blocking.
- Exercises 2: Importance sampling and random walk.
- Exercises 3: Stochastic integration and vanilla option pricing.
- Exercises 4: Symplectic ODE solvers and molecular dynamics.
- Exercises 5 (optional): Hydrogen wavefunction simulation.
