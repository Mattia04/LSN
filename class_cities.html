Detailed Description

represents a collection of cities.

The Cities class contains a vector of City objects, and a vector of possible UIDs (the UIDs of the cities in the collection).

The Cities class contains a map that associates each possible UID to the index of the city in the vector, to optimize data retrieval.

The Cities class also contains a distance matrix, which contains the precomputed distances between all cities for optimization purposes.

The Cities class is designed to be immutable.
